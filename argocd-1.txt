ArgoCD is a deployment tool to deploy applications into kuberentes cluster, maintain the kuberentes cluster..
1. Maintain the cluster
2. Deploy the applications

Advantages
==========
1. no need to install extra tools like kubectl
2. no need to provide authentication to jenkins
3. GitOps --> Git as a single source of truth.
We need to manually push the YAML files.
4. ArgoCD can track the changes in git repos and actively sync into kuberentes cluster.
5. revert to old version. if application is failed, raise another PR to previous PR.
6. Accidental changes in EKS cluster, will be automatically raise since argocd continously syncing the github repos
7. clean UI
